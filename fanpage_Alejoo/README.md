# Fan Page de Bad Bunny

Este proyecto es una fan page sencilla de Bad Bunny, creada como ejercicio académico para practicar:

- HTML básico  
- CSS básico  
- Control de versiones en Git  
- Manejo de ramas (main, desarrollo, release)  
- Despliegue con GitHub Pages  

## Contenido de la página

La página incluye:

- **Header** con título, subtítulo y menú.  
- **Body** con texto, imágenes y botones que redireccionan a Spotify y YouTube del artista.  
- **Footer** con dirección, correo, teléfono y botón hacia Google Maps.

## Ramas del repositorio

- `main` → versión estable final  
- `desarrollo` → rama de trabajo  
- `release` → rama para pruebas previas al despliegue  

## Cómo visualizar la página

Este proyecto será desplegado en **GitHub Pages**.
